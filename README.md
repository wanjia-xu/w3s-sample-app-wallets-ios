## Installation

1. Clone this repo
2. Open project folder `$ cd w3s-sample-app-wallets-ios`
3. Run `$ pod install` to install `CircleProgrammableWalletSDK_Internal`
4. Run `$ pod update` to update SDK (Optional)
5. Open the `.xcworkspace` file in Xcode
